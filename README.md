# dotfiles

Some quick files to get my OS X development environment running.

## Installation

```bash
(mkdir -p /tmp/dots && cd /tmp/dots && curl -L https://github.com/qjroberts/dotfiles/archive/master.tar.gz | tar zx --strip 1 && ./startup.sh)
```
